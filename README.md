# Bargain Finder AI Chatbot

A responsive web application that uses Google's Gemini AI to help users find the best prices for products across various e-commerce platforms.

## Features

- Modern, responsive UI design
- Real-time price comparison using Gemini AI
- Support for multiple e-commerce platforms
- Detailed product information and specifications
- Loading animations for better user experience

## Setup Instructions

1. Clone this repository to your local machine
2. Get a Gemini API key from [Google AI Studio](https://makersuite.google.com/app/apikey)
3. Open `script.js` and replace `YOUR_GEMINI_API_KEY` with your actual API key
4. Open `index.html` in your web browser or serve it using a local web server

## Usage

1. Open the application in your web browser
2. Type your product search query in the input field
3. Press Enter or click the Search button
4. The AI will analyze your request and provide:
   - Product specifications
   - Price comparisons from different platforms
   - Links to the best deals
   - Information about ongoing discounts

## Example Queries

- "Find the best price for iPhone 15 Pro"
- "Compare prices for Sony WH-1000XM5 headphones"
- "What's the cheapest place to buy a MacBook Pro M2?"
- "Find deals on Samsung Galaxy S23 Ultra"
